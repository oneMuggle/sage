## 12.9 风险评估

| 风险          | 影响 | 应对                  |
| ------------- | ---- | --------------------- |
| WebView2 安装 | 高   | 打包引导程序          |
| Win7 TLS 问题 | 高   | 预装证书/备选方案     |
| ChromaDB 性能 | 中   | 使用 SQLite FTS5 替代 |
| Python 兼容性 | 中   | 限定 3.8-3.11         |

## 12.10 质量标准

### 12.10.1 代码质量

- [ ] 类型覆盖率 > 80%
- [ ] 单元测试覆盖率 > 60%
- [ ] ESLint 无 error
- [ ] 无安全漏洞

### 12.10.2 性能标准

- [ ] 启动时间 < 3 秒
- [ ] 空闲内存 < 100MB
- [ ] 对话响应 < 5 秒 (不计 AI)
- [ ] 内存增长 < 50MB/天

### 12.10.3 兼容性

- [ ] Windows 7 SP1
- [ ] Windows 10
- [ ] Windows 11
